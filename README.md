# Phone2.xyz

Phone2.xyz is the AI Safety Net for home service pros. The one-page marketing site explains the multi-user SaaS, the manager dashboard, client portal, onboarding, and ROI calculator for contractors who rely on Thumbtack and Angi leads.

## Tech
- Static HTML with Tailwind CSS via CDN
- Light-weight interactive ROI calculator powered by vanilla JavaScript
- Deployable on Vercel or any static host

## Deploy
```bash
cd /home/strid3r/.openclaw/workspace/phone2-xyz
vercel --prod
```

The site illustrates the Safety Net (missed call text-back), Lead Dialer, and the Phone2 way of protecting every lead so contractors never buy the same lead twice.
